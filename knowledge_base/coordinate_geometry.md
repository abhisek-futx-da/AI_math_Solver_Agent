# Coordinate Geometry (JEE Level)

## Straight Lines
- Distance between (x₁,y₁) and (x₂,y₂): d = √[(x₂−x₁)² + (y₂−y₁)²]
- Mid-point: ((x₁+x₂)/2, (y₁+y₂)/2)
- Slope: m = (y₂−y₁)/(x₂−x₁)
- Slope-intercept form: y = mx + c
- Point-slope form: y − y₁ = m(x − x₁)
- Two-point form: (y−y₁)/(y₂−y₁) = (x−x₁)/(x₂−x₁)
- Distance from point (x₀,y₀) to line ax+by+c=0: |ax₀+by₀+c| / √(a²+b²)

## Circle
- Standard form (centre origin): x² + y² = r²
- General form (centre (h,k)): (x−h)² + (y−k)² = r²
- General equation: x² + y² + 2gx + 2fy + c = 0; centre=(−g,−f), r=√(g²+f²−c)

## Parabola
- y² = 4ax: focus (a,0), directrix x = −a, vertex (0,0)
- x² = 4ay: opens upward

## Ellipse (a > b)
- x²/a² + y²/b² = 1; foci (±ae, 0) where e = √(1−b²/a²)
- Sum of distances from foci = 2a

## Hyperbola
- x²/a² − y²/b² = 1; foci (±ae, 0) where e = √(1+b²/a²) > 1
- Asymptotes: y = ±(b/a)x

## Section formula (internal division in ratio m:n)
- x = (mx₂ + nx₁)/(m+n),  y = (my₂ + ny₁)/(m+n)
