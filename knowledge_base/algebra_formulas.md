# Algebra - Core Formulas

## Quadratic equations
- Standard form: ax² + bx + c = 0
- Roots: x = (-b ± √(b² - 4ac)) / (2a)
- Discriminant D = b² - 4ac: D > 0 two distinct roots, D = 0 repeated root, D < 0 no real roots
- Sum of roots = -b/a, Product of roots = c/a

## Linear equations
- ax + b = 0 ⇒ x = -b/a (a ≠ 0)
- Two variables: a₁x + b₁y = c₁, a₂x + b₂y = c₂. Use substitution or elimination.

## Exponents and logarithms
- a^m · a^n = a^(m+n); a^m / a^n = a^(m-n); (a^m)^n = a^(mn)
- log(ab) = log a + log b; log(a/b) = log a - log b; log a^n = n log a
- Domain: for log x, x > 0

## Inequalities
- Multiplying/dividing by negative reverses inequality
- |x| < a ⇒ -a < x < a; |x| > a ⇒ x < -a or x > a
