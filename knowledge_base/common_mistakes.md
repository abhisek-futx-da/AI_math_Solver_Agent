# Common Mistakes and Pitfalls

## Algebra
- Dividing by zero: never divide by an expression without checking it is non-zero (e.g. when cancelling (x-1)).
- Squaring: if a² = b² then a = ± b; extraneous roots when squaring both sides.
- Domain: √(x) requires x ≥ 0; 1/x requires x ≠ 0; log x requires x > 0.

## Probability
- P(A|B) ≠ P(B|A) in general; use Bayes to convert.
- "At least one" = 1 - P(none). Don't double-count in inclusion-exclusion.
- Replacement vs no replacement: P(second red | first red) changes.

## Calculus
- Chain rule: derivative of f(g(x)) needs both f'(g(x)) and g'(x).
- Domain: ln(x) only for x > 0; critical points must lie in domain.
- Optimization: always check boundary of domain (e.g. closed interval).

## Linear algebra
- Matrix multiplication is not commutative: AB ≠ BA in general.
- Determinant of 2×2: ad - bc (minus sign on bc).
- When inverting, verify det(A) ≠ 0.
