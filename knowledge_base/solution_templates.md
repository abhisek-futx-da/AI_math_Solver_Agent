# Solution Templates

## Quadratic equation template
1. Write in form ax² + bx + c = 0.
2. Compute discriminant D = b² - 4ac.
3. If D ≥ 0: x = (-b ± √D)/(2a). If D < 0: no real roots (or state complex roots if required).

## Probability word problem template
1. Define events clearly (e.g. A = "first draw red").
2. Identify whether order matters and with/without replacement.
3. Use tree diagram or formula: P(A and B) = P(A)P(B|A) or counting.
4. Check: probabilities sum to 1 where applicable; 0 ≤ each ≤ 1.

## Derivative / optimization template
1. Write function f(x) and state domain (e.g. x > 0).
2. Find f'(x) and set f'(x) = 0 for critical points.
3. Check endpoints of domain if bounded.
4. Compare values at critical points and endpoints; state max/min and where.

## Linear system template
1. Write augmented matrix [A|b].
2. Row reduce to REF/ RREF.
3. Identify free variables; express leading variables in terms of them.
4. State "unique solution", "infinitely many", or "no solution".
