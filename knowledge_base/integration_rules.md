# Integration - Rules and Standard Results (JEE Level)

## Basic rules
- ∫ xⁿ dx = xⁿ⁺¹ / (n+1) + C   (n ≠ −1)
- ∫ 1/x dx = ln|x| + C
- ∫ eˣ dx = eˣ + C
- ∫ aˣ dx = aˣ / ln a + C
- ∫ sin x dx = −cos x + C
- ∫ cos x dx = sin x + C
- ∫ sec²x dx = tan x + C
- ∫ cosec²x dx = −cot x + C
- ∫ sec x tan x dx = sec x + C
- ∫ cosec x cot x dx = −cosec x + C

## Standard results
- ∫ 1/√(1−x²) dx = arcsin x + C
- ∫ 1/(1+x²) dx = arctan x + C
- ∫ 1/(x²−a²) dx = (1/2a) ln|(x−a)/(x+a)| + C
- ∫ 1/√(x²+a²) dx = ln|x + √(x²+a²)| + C

## Integration by parts (ILATE order: Inverse, Log, Algebraic, Trig, Exponential)
- ∫ u dv = uv − ∫ v du

## Definite integral properties
- ∫ₐᵇ f(x) dx = −∫ᵦₐ f(x) dx
- ∫ₐᵇ f(x) dx = ∫ₐᶜ f(x) dx + ∫ᶜᵦ f(x) dx
- ∫₋ₐᵃ f(x) dx = 2∫₀ᵃ f(x) dx  if f is even;  0 if f is odd
- ∫₀ᵃ f(x) dx = ∫₀ᵃ f(a−x) dx  (substitution rule)

## Substitution method
- For ∫ f(g(x)) g'(x) dx: let u = g(x), du = g'(x) dx → ∫ f(u) du
